# 必看说明

## 1.关于SSR.

本站的服务基于原版SS协议，理论上也支持SSR协议。

由于SSR项目停止维护，其衍生版本多，存在不同的订阅格式，我们无法全部兼容。

现阶段SSR的过多参数反而成了“特征”，相比之下原版SS更加可靠。

{% hint style="success" %}
**本站推荐使用SHADOWSOCKS原版软件**
{% endhint %}
